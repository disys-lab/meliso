# InMemMVM

This repository contains a fork of [**MLP_Neurosim_V3.0**](https://github.com/neurosim/MLP_NeuroSim_V3.0), a device–circuit–algorithm
framework originally developed at Arizona State University (ASU) and Georgia Institute of Technologu 
for benchmarking neuro-inspired architectures with synaptic devices
(e.g., SRAM and emerging non-volatile memory).

## About This Fork

This repository is a research fork of NeuroSim maintained by:

**Distributed Intelligent Systems (DISys) Laboratory**  
Oklahoma State University  

This fork may contain modifications and extensions made in 2026 to
support scalable computation beyond the original architectural constraints.

The purpose of this fork is to support ongoing academic research in
large-scale optimization, device–algorithm co-design, and related topics.
